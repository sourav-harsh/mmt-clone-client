# MakeMyTrip Clone

A production-ready frontend clone of MakeMyTrip built with React, Vite, Tailwind CSS, and React Icons.

## Live Demo : [MakeMyTrip Clone](https://mmt-clone-client.vercel.app/)

![Hotel Search Page](/public/images/Hotel_search_page.png)

## Project Structure

```
src/
  pages/
    Home/
      Home.jsx
      components/
    Flights/
      Flights.jsx
      components/
    Hotels/
      Hotels.jsx
      components/
    Holidays/
      Holidays.jsx
      components/
    Trains/
      Trains.jsx
      components/
  utils/        # Common components (Navbar, Footer, etc.)
public/
  images/       # Static images
```

## Setup

```bash
npm install
npm run dev      # start dev server
npm run build    # production build
npm run preview  # preview build
```

## Tech

- React 18
- React Router v6
- Tailwind CSS 3
- React Icons
- Vite 5
